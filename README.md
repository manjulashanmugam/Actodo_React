# 🚀 Actodo – React Todo Application

Actodo is a simple and clean Todo application built using React.  
It allows users to sign up, log in, and manage their daily tasks efficiently.

## ✨ Features

- 🔐 User Signup & Login
- ➕ Add new todos
- ❌ Delete todos
- 📋 View task list
- 🎨 Clean UI with Tailwind CSS
- 🔄 React Router navigation

## 🛠️ Built With

- React JS
- React Router DOM
- Tailwind CSS
- JavaScript (ES6)

## 📂 Project Structure

src/
 ├── components/
 ├── pages/
 ├── App.js
 ├── index.js


## ▶️ How to Run Locally

1. Clone the repository

git clone https://github.com/manjulashanmugam/Actodo_React.git

2. Install dependencies

npm install

3. Start the development server

npm start
